Predicting Gawker Spring 2015

Machine Learning

As part of a semester long class project, our team researched which characteristics of Gawker articles yield greater article popularity than others. We gathered tens of thousands of Gawker articles and computed article features to explore this question. During our investigation we used data analytics software and machine learning to find potential correlations and created a classifier that will predict an articles popularity given an article's features. Basic features include, an article's title, author, and picture count. Our classifier reached an F1 score of 0.51. We hope that our work can can be the foundation for further research and exploration. The project uses Ruby for obtaining the dataset of articles and Python (specifically Scikit-learn) for machine learning.

Security-Aware Synaptic Spring 2015

Linux Package Manager

Worked with a team of 8 in the Software Engineering course at the University of Maryland to define and build a product that meets a client's needs. The client requested a security-aware package manager that allows the user to make informed decisions regarding the hidden security risks when installing and updating packages. Our team used Synaptic, a visual package manager based on APT for Debian, as a foundation for this project. We expanded its GUI capabilities to show security information such as up-to-date package vulnerability data, package risk scores, and potential change in risk when updating a package. Modifications to existing source code were made in C++, and preprocessing of security data was done with Ruby.
